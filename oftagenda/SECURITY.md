# Política de Segurança

Este arquivo define como reportar vulnerabilidades no `oftagenda`.

## Versões suportadas

| Versão | Suporte |
| --- | --- |
| `main` | :white_check_mark: |
| outras branches | :x: |

## Como reportar

Envie o relato de forma **privada** para `contato@oftleonardo.com.br` com o assunto:

`[SECURITY][oftagenda] <resumo da vulnerabilidade>`

Inclua:

- descrição e impacto;
- passos de reprodução;
- rotas/endpoints e versão/commit;
- evidências (logs, prints, payloads, PoC).

## SLA de resposta

- confirmação de recebimento: até 3 dias úteis;
- primeira análise: até 7 dias úteis;
- atualizações periódicas até a resolução.

Não abra issue pública para vulnerabilidades de segurança.
